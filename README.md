# Motivation-WHY

* Tăng Cường Khả Năng Tiếp Cận Giáo Dục
* Cá Nhân Hóa Quá Trình Học Tập
* Linh Hoạt Về Thời Gian Và Địa Điểm
* Cải Thiện Chất Lượng Giáo Dục
* Giảm Áp Lực Học Tập
